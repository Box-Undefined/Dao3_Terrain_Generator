# 神奇代码岛（*dao3.fun*）地形生成
- 可自定义地形，**无算法，无种子**
- 可以直接复制到控制台
- 地形数据使用***gat_MAP_info.py***运行文件为***main.js***
- 运行时，请在*dao3.fun/e/* 或 *goboxgame.com/e/*上运行

## 用法说明
1. **准备**
   准备如图的png图片*map.png* (即图片尺寸为地图的*X*×*Z*，图片为灰度图)
   
   ![地形图](/map.png)
   你可以选择像上图那样做有层次感的地图，也就是说，在绘制图片时使用二值笔（PhotoShop铅笔工具）绘制
   ##### 用PhotoShop绘制
   1. 新建一个长为地图尺寸x，宽为地图尺寸z的空白图片
   2. 使用调色板上的纯灰色，纯白色或纯黑色绘制地图，纯黑色(*#000000*)为地图最低处，纯白色(*#FFFFFF*)为地图最高点，用**铅笔工具**绘制地图
   3. 使用文件--导出--**快去导出为PNG**进行导出（注意一定是快速导出为PNG，不然地形会变得坑坑洼洼）导出到与*gat_MAP_info.py*和*main.js*同一目录下
   ##### 用画世界Pro绘制
   1. 新建一个长为地图尺寸x，宽为地图尺寸z的**像素画板**
   2. 使用调色板上的纯灰色，纯白色或纯黑色绘制地图，纯黑色(*#000000*)为地图最低处，纯白色(*#FFFFFF*)为地图最高点，用**画笔--二值笔**绘制地图
   3. 导出为PNG并重命名为*map.png*
      ***
   当然你也可以选择制作平缓的地形，例如PS--滤镜--渲染--云，或者按照如上操作后添加 高斯模糊 滤镜；
